# MOOC Java Programming I

This repository contains my work for the MOOC Java Programming I course. The MOOC Java Programming I course is an online course that provides an introduction to Java programming. It covers topics such as basic programming concepts, control structures, arrays, and object-oriented programming.

## Course Information

- Course name: MOOC Java Programming I
- Course provider: University of Helsinki
- Course website: https://java-programming.mooc.fi/

## Course Content

The MOOC Java Programming I course consists of seven parts, each with several chapters and exercises:

1. Basics of programming
2. Conditional statements and loops
3. Arrays
4. Basic data structures
5. Object-oriented programming, part 1
6. Object-oriented programming, part 2
7. Miscellaneous

The course provides a good foundation in Java programming concepts and techniques. The exercises are challenging and require a good understanding of the course material.

## Repository Contents

This repository contains my solutions to the programming exercises for the MOOC Java Programming I course. Each part of the course has its own folder, and each folder contains subfolders for each chapter of the course. Within each chapter folder, there are folders for each exercise. Each exercise folder contains the code for my solution to the exercise.
